Introduction

This portfolio showcases my progression from Graduate to Proficient teacher using the CECE framework. Each evidence set demonstrates how my planning, pedagogy, assessment practices, and professional engagement align with the Australian Professional Standards for Teachers.

Use this site as a hub to explore six curated evidence sets, each with context, claims, reflections, and linked artifacts stored in Google Drive.

Portfolio Overview

  • Evidence Set 1: Year 9 Food Security portfolio assessment with weekly checkpoints and outcomes tracking.
  • Evidence Set 2: Parent communication and participation rubric to lift Year 10 HASS engagement.
  • Evidence Set 3: Year 7 HPE Parntu invasion games unit combining cultural learning and movement skills.
  • Evidence Set 4: Engagement rubric and family communication artifacts supporting classroom expectations.
  • Evidence Set 5: ICT integration through a WWII interactive story using CoSpaces and Inklewriter.
  • Evidence Set 6: Task modification of a Year 10 Environmental Management Plan into a scaffolded action plan.

How to Navigate
